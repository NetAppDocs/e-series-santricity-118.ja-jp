:allow-uri-read: 
:icons: font

[role="lead"]
[実行中の処理]ページでは、長時間実行されている処理のうち、現在実行中、中断、停止、または失敗した状態にある処理のリストを表示できます。処理のタイプによっては対処が可能です。

長時間の処理がシステムパフォーマンスに影響する可能性があります。一部の処理では、処理を停止したり、重要度の高い処理を先に完了するように優先度を変更したりできます。
