= Snapshotにはどのような種類のボリュームを使用できますか。
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Snapshotイメージの格納に使用できるボリュームは、標準ボリュームとシンボリュームだけです。標準以外のボリュームは使用できません。ベースボリュームはプールまたはボリュームグループのいずれかに配置できます。
