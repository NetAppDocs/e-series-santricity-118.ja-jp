= 個 々 のストレージアレイの管理
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
[起動]オプションを使用すると、管理処理を実行する場合に1つ以上のストレージアレイに対してブラウザベースのSystem Managerを開くことができます。

.手順
. [管理]ページで、管理するストレージアレイを1つ以上選択します。
. [* 起動 * ] をクリックします。
+
新しいウィンドウが開き、System Managerのログインページが表示されます。

. ユーザー名とパスワードを入力し、*ログイン*をクリックします。
